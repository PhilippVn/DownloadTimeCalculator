# DownloadTimeCalculator
calculate remaining download time for your favourite movies, games, ...

# Requirements:
- gradle
- java 17

# Compile and Run
from root directory:
- using gradle wrapper: `./gradlew run`
- jar:
  - compile: `./gradlew build`
  - run: `java -jar <path to jar>` (usually app/build/libs/<jarname>)
